Heading (H1)
Alt text
Paragraph
Alt text is a property of an image that allows screen readers to describe the image.
Paragraph
Guidelines for alt text:
List (unordered)
  • Concise
  • Include any relevant text in the image
  • Reflects intention and context of the image
  • May be left blank if image is purely decorative or redundant
  • Confirm any auto generated alt text
